쇼핑몰 제작

Spring Framwork을 이용하여 쇼핑몰을 제작하였다.


사용 언어

#Spring Framwork #JAVA(Servlet, JSP) #SQL Developer #Mybatis(3.4.6) #Tomcat(8.5)


페이지 구성

- 로그인, 회원가입, 회원정보보기 페이지, 장바구니, 주문정보 확인, 카페고리별 상품보기를 구현하였다.
- 회원 ID와 PASSWD, 회원정보, 장바구니 등의 정보는 oracle 11g SQL Developer를 연동하였다.
- 회원 ID 세션 관리, Ajax 비동기 처리, 데이터 중복 여부 확인 등의 기능을 사용하였다.
