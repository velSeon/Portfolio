미니 블로그 사이트

파이썬 언어를 기반으로 Flask 라이브러리를 이용해서 기본적인 블로그 서비스를 구축하였다.


사용 언어

#Python3 #Flask #SQLite3 #HTML5 #CSS3 #Bootstrap


페이지 구성

- admin page(관리자 페이지)를 구성하여 쉽게 블로그 글을 수정, 삭제, 사진 파일 업로드를 할 수 있다.
- contact 페이지를 이용하여 외부에서 자신의 메일로 연락을 확인 할 수 있다.
- bootstrap을 이용하여 사용자가 편하게 볼 수 있게 반응형 웹사이트로 제작하였다.


본인 역할

1. 메인 페이지에 포스팅한 글의 갯수를 조정하여 페이징(PREVIOUS, NEXT 버튼을 이용하여 이동할 수 있다.)
2. 포스팅 글을 클릭 시 해당 포스팅으로 페이지 이동할 수 있게 하였다.
3. 관리자 페이지에서 글 수정, 삭제, 사진 파일 업로드(template>static 폴더로 업로드), 메인페이지 이동 버튼을 구현하였다.
4. bootstrap을 활용하여 반응형 웹사이트를 구현하였다.
